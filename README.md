# Calculadora-Obra-Residencial
API REST desenvolvida em Java com Spring Boot para o cálculo automatizado de insumos (concreto para vigas baldrames e quantitativo de tijolos). O projeto utiliza modelagem de Grafos para representar a planta baixa e persistência de dados com Spring Data JPA (Hibernate). Inclui análise de Pontos de Função e geração de relatórios em PDF.
